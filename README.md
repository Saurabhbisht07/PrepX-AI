PrepX вЂ“ AI Powered Interview Preparation Platform

PrepX is a full-stack AI-powered interview preparation platform that helps users practice interviews, generate questions from resumes, and receive intelligent feedback.

The platform allows users to upload their resume, practice technical and HR interviews, and improve their performance through AI-generated questions and feedback. It also includes a credit-based system with Razorpay payments, making it similar to a real-world SaaS product.

This project demonstrates production-level full-stack development including authentication, payments integration, backend APIs.
