PrepX – AI Powered Interview Preparation Platform

PrepX is a full-stack AI-powered interview preparation platform that helps users practice interviews, generate questions from resumes, and receive intelligent feedback.

The platform allows users to upload their resume, practice technical and HR interviews, and improve their performance through AI-generated questions and feedback. It also includes a credit-based system with Razorpay payments, making it similar to a real-world SaaS product.

This project demonstrates production-level full-stack development including authentication, payments integration, backend APIs.


✨ Features

✅ Upload Resume (PDF)
✅ AI Generated Interview Questions
✅ Practice Technical & HR Interview Rounds
✅ Intelligent Feedback System
✅ Credit-Based Usage System
✅ Purchase Credits using Razorpay
✅ Google Authentication using Firebase
✅ Smooth UI Animations with Framer Motion
✅ Full Stack SaaS Architecture
✅ Deployed on Render

🧠 How It Works

User logs in using Google Authentication (Firebase)

User uploads their resume (PDF)

AI generates interview questions based on resume

User practices technical and HR interview rounds

System provides feedback and improvement suggestions

Users spend credits to access interview sessions

Credits can be purchased through Razorpay payment gateway

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/Saurabhbisht07/PrepX-AI.git
2️⃣ Go to Project Folder
cd PrepX-AI
3️⃣ Install Dependencies

Frontend:

cd client
npm install

Backend:

cd server
npm install
4️⃣ Environment Variables

Create .env file in the server folder

Example:

MONGO_URI=your_mongodb_connection
RAZORPAY_KEY_ID=your_key
RAZORPAY_SECRET=your_secret
FIREBASE_CONFIG=your_firebase_config
5️⃣ Run the Project

Backend:

npm run server

Frontend:

npm start
🌐 Deployment
